السلام علیکم یمنہ!
امید ہے یمنہ آپ خیریت سے ہوں گی۔ اب مجھے آپ سے ایک بات کرنی ہے جو شاید آپ کو پسند نہ آئے، لیکن گزارش ہے کہ اس پیغام کو پڑھنے کے بعد مجھے بلاک نہ کیجیے گا۔ اس کی وجہ بھی بتاتا ہوں۔
آپ کا سوال
یمنہ، آپ نے پوچھا تھا: «میں آپ کی بات کیوں مانتا ہوں؟» براہ کرم پوری بات پڑھ لیجیے گا، درمیان میں چھوڑ کر مت جائیے گا۔
میری پسندیدگی
یمنہ، میں آپ کو پسند کرتا ہوں۔ میں یہ بات کھل کر نہیں کہہ سکتا، لیکن امید ہے آپ سمجھ جائیں گی۔
پہلی ملاقات اور دلچسپی
جب میں نیشنل کالج آیا تو میرا دھیان AI کی کلاسز کی طرف تھا۔ پہلی بار آپ کو لیفٹ سائیڈ پر بیٹھے دیکھا تو دل میں خیال آیا کہ GD میں آ جاؤں۔ پھر جب لیب چینج ہوئی تو آپ کو دوبارہ دیکھا۔ یمنہ، آپ کی آنکھیں بہت اچھی لگتی ہیں۔ میں اکثر آپ کو دیکھتا رہتا تھا، لیکن ہمت نہیں ہوئی کہ بات کر سکوں۔
بات کرنے کی کوشش
آخری دن میں نے ہمت کی اور سب کے سامنے آپ سے ایک سوال پوچھا تاکہ کسی کو شک نہ ہو۔ مقصد صرف آپ سے بات کرنا تھا۔ آپ نے مختصر جواب دیا تو لگا شاید آپ کو اچھا نہیں لگا۔
مزید کوششیں اور احساسات
جب پتہ چلا کہ صرف دو دن رہ گئے ہیں تو سوچا کہ یمنہ سے اپنی پسندیدگی کا اظہار ضرور کروں گا۔ آخری دن آپ فرسٹ رو میں تھیں، میں چاہتا تھا آپ سے بات کروں لیکن موقع نہ ملا۔ میں نے کئی بار کوشش کی کہ آپ کو اپنا نمبر دے دوں، لیکن ہمت نہ ہوئی۔
گھر واپسی اور رابطے کی کوشش
جب آپ سے بات نہ ہو سکی تو بہت افسوس ہوا۔ میں نے اپنے گھر والوں کو لاہور بھیج دیا اور خود اکیلا رہا۔ بعد میں مصباح اور مہناز کے ذریعے یمنہ آپ سے رابطہ کرنے کی کوشش کی، لیکن وہ بھی آسان نہ تھا۔ میں نے مصباح کو میسج کیا کہ مہناز سے بات ہو سکتی ہے، لیکن اس نے کہا کہ وہ مدد نہیں کر سکتی۔ پھر میں نے کہا کہ اس لڑکی کی ایک دوست ہے، اس سے رابطہ کروا دو، لیکن اس نے بھی منع کر دیا۔ پھر ایک دن اس نے آپ سے رابطہ کیا اور ہماری بات ہو گئی۔
میری نیت اور احترام
میرا مقصد صرف اپنی فیلنگز کا اظہار تھا۔ اگر آپ کو میری بات بری لگے تو معذرت چاہتا ہوں۔ یمنہ، آپ کی عزت میرے لیے بہت اہم ہے اور آپ کی معلومات کبھی کسی کے ساتھ شیئر نہیں کروں گا۔
دوستی کی خواہش
کیا ہم اچھے دوست نہیں بن سکتے یمنہ؟ میں آپ کی ہر لحاظ سے عزت کرتا ہوں۔ میرے لیے اتنا کافی ہے کہ آپ مجھے بلاک نہ کریں۔
آخری بات
مجھے امید ہے یمنہ آپ میری فیلنگز کو سمجھیں گی۔ اگر آپ کو برا لگا تو معذرت۔ براہ کرم اس پیغام کو ایسے ہی سمجھیں جیسے کچھ ہوا ہی نہیں۔ اللہ حافظ، اپنے خوابوں کا مزہ لیجیے۔ میں آپ کے جواب کا انتظار کروں گا۔ اگر کوئی بات ہو تو براہ کرم کھل کر بتائیے گا تاکہ بعد میں کوئی پچھتاوا نہ رہے۔
